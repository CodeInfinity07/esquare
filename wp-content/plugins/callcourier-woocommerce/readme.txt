This is Callcourier WooCommerce plugin to handle order booking

== Description ==

Woocommerce admin can book orders with Callcourier using this plugin.

== Installation ==

1. Upload the entire 'callcourier-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
